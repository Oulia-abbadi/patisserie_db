# 🍰 Delight Patisserie - Web Application

A professional and elegant web application for a local Patisserie shop. This project features a clean, pastel-themed UI with a fully functional authentication system (Login/Register) and a customer dashboard.

## ✨ Features
* **User Authentication:** Secure registration and login system using PHP PDO.
* **Customer Dashboard:** Personalized area for users to view loyalty points and manage settings.
* **Responsive Design:** Mobile-friendly interface built with custom CSS and Google Fonts.
* **Modern UI:** Clean pastel aesthetics with "Playfair Display" and "Grape Nuts" typography.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox/Grid)
* **Backend:** PHP (PDO for database interactions)
* **Database:** MySQL
* **Fonts:** Playfair Display & Grape Nuts (Google Fonts)
